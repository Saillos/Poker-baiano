📖 LIVRO DE REGRAS

O Bettle Trinca é um jogo de estratégia e antecipação para dois jogadores. O objetivo é vencer a disputa em 3 das 5 colunas da mesa, formando as melhores combinações de 3 cartas (Trincas).

1. O Baralho

  • Total: 54 cartas (Baralho padrão de 52 cartas + 2 Coringas).
  • Coringas (JK): São cartas "wild". Elas assumem automaticamente o valor e naipe necessários para formar a maior combinação possível na coluna onde forem jogadas.

2. Fluxo da Partida

  • Cada jogador começa com 5 cartas na mão.
  • A cada turno, o jogador deve baixar 1 carta em qualquer uma das suas 5 colunas.
  • Imediatamente após jogar, o jogador compra uma nova carta do deck.
  • O turno passa para o oponente.

3. A Coluna Zero (Estratégia)

A primeira coluna (Coluna 0) é especial:

  • As cartas jogadas ali ficam viradas para baixo (ocultas) para o oponente.
  • Elas só são reveladas quando ambos os jogadores completarem 3 cartas nessa coluna ou quando o jogo terminar.

4. Como Vencer uma Coluna

Uma coluna é disputada quando ambos os jogadores colocam 3 cartas nela. A hierarquia de forças (da maior para a menor) é:

  1. Trinca de Ás: Três cartas "A".
  2. Straight Flush: Sequência de 3 cartas do mesmo naipe.
  3. Trinca: Três cartas do mesmo valor.
  4. Flush: Três cartas do mesmo naipe, sem sequência.
  5. Sequência (Straight): Três cartas em ordem numérica.
  6. Par: Duas cartas do mesmo valor.
  7. Carta Alta: Soma dos valores das 3 cartas.
